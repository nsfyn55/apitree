=========
iomanager
=========

``pyramid_apitree`` is builds an orderly web API from Pyramid views.

If you are using Python 2.7.4 or lower, you should use ``pyramid_apitree``
version 0.2.0. If you are using Python 3.x, you should use the most current
version of ``pyramid_apitree``. Python 2.x is no longer supported starting
with ``pyramid_apitree`` version 0.3.0.

For more information about ``pyramid_apitree``, se the project's github
page:
`github.com/jmatthias/pyramid_apitree <https://github.com/jmatthias/pyramid_apitree>`_
